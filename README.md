# zenn-articles

このリポジトリには、aKuad が [Zenn](https://zenn.dev/) に投稿しているコンテンツを格納しています。Zenn の GitHub 連携により、本リポジトリのコンテンツが Zenn コンテンツに反映されています。

It's a repository what containing aKuad's Zenn contents. These contents are deployed by Zenn's GitHub integration.

<https://zenn.dev/akuad>

## 問題の報告・修正リクエスト

コンテンツにタイポや誤情報など、修正すべき問題がありましたら、[Issue](https://github.com/aKuad/zenn-articles/issues/new) で報告、または Pull Request で編集を提案いただけますと幸いです。

Issue では、当該コンテンツの URL または Markdown ファイル名と合わせて、問題内容をご報告ください。また、重複する Issue が既に立てられていないかも事前にご確認ください。
